# Examples Directory

Development examples showcasing Foundation library usage.

## Files

### example.html
Minimal example showing how to use Foundation library functions directly.
- Basic Thiele-Small calculations
- SPL calculations
- Alignment calculations

**View:** http://localhost:3000/examples/example.html

### foundation.html
Interactive showcase of Foundation library (189 tested functions).
- All Small 1972/1973 functions
- All Thiele 1971 alignment functions
- Visual test results
- Function documentation

**View:** http://localhost:3000/examples/foundation.html

## Purpose

These files are for:
- Learning how to use Foundation library
- Verifying Foundation calculations
- Development reference

**Not for production use** - BoxSmith UI (ui/index.html) is the main application.
