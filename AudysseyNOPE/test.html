Configuration

Honestly, few should touch these settings. They are here for the curious and the brave. The default values are chosen to work well with most systems to the best of my knowledge.

Setting Value Description
Speaker Align Frequency 400 Center frequency (in Hz) for speaker level alignments
Speaker Align Span 2 Span in octaves for speaker level alignments, half above and half below the center frequency
Speaker Target Adjustment 1 Speaker pre-filter calculation target level adjustment (in dB)
Subwoofer Align Frequency 60 Center frequency (in Hz) for subwoofer level alignments
Subwoofer Align Span 1 Span in octaves for subwoofer level alignments, half above and half below the center frequency
Subwoofer Target Adjustment -2 Subwoofer pre-filter calculation target level adjustment (in dB)
Minimum Crossover Frequency 80 Minimum crossover frequency (in Hz) to use for any speaker, no matter what is detected
Maximum Crossover Frequency 150 Maximum crossover frequency (in Hz) to use for any speaker, no matter what is detected
Filter Lower Limit, Speakers 20 Lower frequency limit (in Hz) for filter creation arithmetic
Filter Upper Limit, Speakers 250 Upper frequency limit (in Hz) for filter creation arithmetic
Filter Max Gain, Speakers 5 Maximum gain (in dB) any one filter is allowed to apply
Filter Lower Limit, Subs 10 Lower frequency limit (in Hz) for filter creation arithmetic
Filter Upper Limit, Subs 200 Upper frequency limit (in Hz) for filter creation arithmetic
Filter Max Gain, Subs 5 Maximum gain (in dB) any one filter is allowed to apply
System Target Level 75 Target sound pressure level (in dB) for the system
Audyssey Subwoofers Curtain Frequency 250 Frequency (in Hz) to set as Audyssey curtain for subwoofers
Audyssey Speakers Curtain Frequency 350 Frequency (in Hz) to set as Audyssey curtain for speakers
Crossover Detection Tolerance 1 How far below the system target curve (in dB) a response can be at a certain frequency and still count as reaching it
Speaker Pair Level Fixing Limit 1 How far off a speaker pair can be from each other (in dB) for the levels to be adjusted to their mean
